个人博客搭建

Created2024-10-07|Updated2024-10-29
|Post Views:

个人博客搭建过程及心路历程

1. 安装git和node.js

在本地安装git和node.js.

安装时可以直接一路点下一步，按默认设置安装.

此期间出现网站无法打开的问题。下载Watt Toolkit进行加速

验证安装是否成功

win+r输入cmd进入终端，输入node -v、npm -v和git –version

如果出现版本号说明node.js安装成功。

桌面右键

出现Open Git Bash here和Open Git GUI here。则Git安装成功

ps：输入cmd检验时存在卡住不动的问题

此时需要下载一个镜像，我选择的是阿里云，搜索npmmirror镜像站。

在使用说明中直接复制第一个npm install -g cnpm –registry=https://registry.npmmirror.com

win+r输入cmd进入终端，复制进去在n前方加一个c，回车安装

2.下载VS Code

进入官网点击下载，按照指示进行安装

进行VS Code配置

点击左侧下方的拓展，在搜索框中搜索Chinese，安装中文语音包，然后可以正式开始部署

选择存储地点

点击“打开文件夹”。选择要存放代码的地方，这边我将文件夹命名为HEXO

3.安装HEXO主题

初始化文件创造

打开VS Code，进入刚才的HEXO，点击上方栏中的终端，点新建终端，在下方选择GIT BASH。

输入hexo init 加上你想要的名字进行初始化拉取出示文件，这边我要给他命名为wuli 我输入了hexo init wuli。此时出现报错，需要重启电脑。

（忘记截图了借用一下别人的）

此时可能会出现安装卡住的情况，需要手动安装。

先进入刚才创建的文件，输入cd wuli进入。输入cnpm install手动安装

生成基础框架

输入hexo g回车，再输入hexo s，会生成一个网址，ctrl加鼠标左键进入网址，或者复制到浏览器中打开，此时得到一个没有主题的基础框架

更换主题

进入hexo官网，点击主题，这边我选的主题是butterfly

在搜索框中输入butterfly，进入开源地址，拉取代码，复制第一个。

回到VS Code中，ctrl+c停止运行，然后粘贴刚才复制的代码，回车。

安装完成后回到hexo官网，向下滑动找代码安装渲染器。

此时左侧的themes文件夹里已经多出一个butterfly。找到下面的-config-yml,拉倒第九十九行将主题改为butterfly，ctrl+s保存即可运用

在下方输入hexo cl清除，此时我出现报错

(借用图片)
得知缺少组件，打开浏览器搜索hexo-util，下滑找到installation，复制代码进入VS安装，此时报错消失问题解决

生成静态文件

输入hexo g生成静态文件

输入hexo s查看本地效果

进一步完善主题

回到浏览器，在刚才的hexo主题那边找到Docs，点Chinese进入并复制文件名

在根目录中新建文件夹，重命名，将刚才的文件名粘贴进去

进入主题文件夹，将其中代码全部复制，粘贴到刚刚建好的文件夹中，ctrl+s保存

更换主题背景

点左侧的butterfly

点击source

点开img

将想要的图片拖入文件夹

进入-config.buttrefly.yml

在第五十四行输入刚才拖入的图片的名称

保存后即可完成背景更改

更改用户名

点左侧的-config.yml

找到title和author，冒号后面的就是名字可自行更改

将本地文件上传至Github

注册Github账户

打开浏览器搜索Github
点入，根据指引注册账号

新建Github仓库

点击 Create repository

输入仓库名称和描述，勾选Add.gitignore

点击创建仓库即可成功创建仓库

连接Github

打开根目录-config.yml，下划到104行，在type后输入git，再输入仓库地址和分支。仓库地址即刚才新建仓库的链接。

安装hexo部署插件

在下方输入 npm install hexo-deployer-git –save 进行安装

hexo g生成静态文件 hexo d上传至仓库，此时会出现登录框，点击登录授权即可上传

此处我出现报错

原因是没有配置全局邮箱和用户名

输入git config –global user.email “你的邮箱”
git config –global user.name “用户名”进行配置

生成可让他人访问的网页

搜索zeabur，使用Github进行授权，点击开始

新建项目

选择服务

点Github

进行授权后显示仓库

点击进行绑定

随后部署

部署完成后点击网络，点开第一个生成域名。到此博客搭建完成。后续可使用VS将文章上传至仓库，仓库会同步到zeabur。

在博客上发布文章

输入hexo new 加标题即可生成
按md语法写文章后hexo g hexo d上传即可

此处我出现图片无法加载问题

输入cnpm install hexo-asset-img –save下载插件

然后在根目录中将43行的false改成true

不要拼错（doge）
有个人true拼错在电脑前思考了一小时（）
